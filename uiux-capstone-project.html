UI/UX Capstone Project

University of Texas Austin Website Redesign

TEAM PROJECT UI/UX DESIGN USER RESEARCH

Project Overview

Led a comprehensive redesign of the University of Texas Austin website, focusing on improving user experience and accessibility. The project involved extensive user research, prototyping, and testing to create a more intuitive and engaging digital platform.

Research & Discovery

Conducted user interviews, surveys, and usability testing to identify pain points and opportunities for improvement. Created detailed user personas and journey maps to guide the redesign process.

Key Features

Improved Navigation

Streamlined information architecture and intuitive menu structure

Accessibility

WCAG 2.1 compliant design with enhanced readability

Mobile Optimization

Responsive design with mobile-first approach

Content Strategy

Clear hierarchy and improved content organization

Design Process

Wireframing

Created low-fidelity wireframes to establish the new information architecture

Prototyping

Developed high-fidelity prototypes for user testing

Visual Design

Established a cohesive visual design system
