AYSA Kent Academy Website

A comprehensive web platform for youth soccer academy

TEAM PROJECT UI/UX DESIGN WEB DEVELOPMENT

Project Overview

As part of a team at the University of Washington, we designed and developed a comprehensive website for AYSA Kent Academy, a youth soccer organization. The project involved user research, UI/UX design, and full-stack development to create an engaging platform for players, parents, and coaches.

Research & Discovery

We conducted interviews with stakeholders, including coaches, parents, and players, to understand their needs and pain points. This research informed our design decisions and feature prioritization.

UI/UX Design Process

Wireframing

Created low-fidelity wireframes to establish the information architecture and user flow.

Prototyping

Developed interactive prototypes in Figma to test and refine the user experience.

Visual Design

Established a cohesive visual design system with custom illustrations and animations.

Development

Technologies Used

  • Frontend: HTML5, CSS3, JavaScript, React
  • Backend: Node.js, Express
  • Database: MongoDB
  • Deployment: AWS

Key Features

Player Registration

Streamlined registration process with secure payment integration

Team Management

Comprehensive tools for coaches to manage teams and schedules

Parent Portal

Dedicated space for parents to track progress and communicate

Mobile Responsive

Fully responsive design for all device sizes

Results & Impact

User Engagement

Increased by 45% after launch

Registration Efficiency

Reduced registration time by 60%

User Satisfaction

95% positive feedback from stakeholders
